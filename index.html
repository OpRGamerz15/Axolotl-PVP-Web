⚡ About Axolotl PVP+

Axolotl PVP+ is a lightweight Minecraft modpack made for smooth and competitive PvP. It removes unnecessary features and focuses on high FPS, clean visuals, and combat responsiveness.

🚀 Key Features

⚔️ PvP Optimized
Everything tuned for fast combat.
📈 High Performance
Mods to boost FPS and reduce lag.
🎯 Clean UI
Better visibility, less clutter.
🧩 Fabric Based
Built on Fabric for best performance.

📦 Installation

  1. Install Fabric Loader.
  2. Download the modpack from Modrinth.
  3. Launch via Modrinth App, Prism, or ATLauncher.
  4. Start playing.

❓ FAQ

Does it support survival?

No. It is PvP focused.

Which versions work?

Check the Modrinth download page for the latest supported version.

Low FPS?

Allocate 2–4GB RAM, disable shaders, update GPU drivers.

FAQ

How do I install Axolotl PVP+?
Install via Modrinth App, Prism Launcher, or ATLauncher by searching "Axolotl PVP+". For SKLauncher, follow our video tutorial.
What Minecraft version does it support?
Check the Modrinth page for the latest supported version. We keep the pack updated with new MC releases.
Can I use this on servers?
Yes! All mods are client-side and allowed on most competitive PVP servers.
My FPS is still low — what do I do?
Allocate at least 4GB of RAM, disable shaders in-game, and keep your GPU drivers up to date.
I found a bug. How do I report it?
Open an issue on our GitHub page or post in Discord.
