⚡ About Axolotl PVP+

Axolotl PVP+ is a lightweight Minecraft modpack.

It is made for smooth and competitive PvP.

It removes unnecessary features.

It focuses on high FPS and clean visuals.

It improves combat responsiveness.

It is best for duels and PvP servers.

🚀 Key Features

⚔️ PvP Optimized
Everything is tuned for fast combat.
📈 High Performance
Includes mods to boost FPS and reduce lag.
🎯 Clean UI
Improves visibility and removes clutter.
🧩 Fabric Based
Built on Fabric for better performance.
💡 Lightweight
Only essential mods are included.

📦 Installation Guide

  1. Install Fabric Loader.
  2. Download the modpack.
  3. Move files to .minecraft folder.
  4. Launch Minecraft using Fabric profile.
  5. Start playing.

⚙️ Recommended Settings

  • Graphics: Fast
  • Render Distance: 6–10 chunks
  • Smooth Lighting: OFF
  • VSync: OFF
  • Use fullscreen mode.

🧠 Performance Tips

  • Close background apps.
  • Allocate 2–4GB RAM.
  • Update drivers.
  • Use shaders only if PC supports it.

❓ FAQ

Does it support survival?

No. It is PvP focused.

Can I add mods?

Yes, but performance may drop.

Which versions work?

Check the download page.

Does it work on low-end PC?

Yes.

🛠️ Troubleshooting

Game not launching:

  • Check Fabric installation.
  • Check file placement.

Low FPS:

  • Lower settings.
  • Remove shaders.
  • Close apps.

Crashes:

  • Check mod conflicts.
  • Reinstall pack.
  • Use correct version.

FAQ

How do I install Axolotl PVP+?
Install via Modrinth App, Prism Launcher, or ATLauncher by searching "Axolotl PVP+". For SKLauncher, follow our video tutorial.
What Minecraft version does it support?
Check the Modrinth page for the latest supported version. We keep the pack updated with new MC releases.
Can I use this on servers?
Yes! All mods are client-side and allowed on most competitive PVP servers.
My FPS is still low — what do I do?
Allocate at least 4GB of RAM, disable shaders in-game, and keep your GPU drivers up to date. Check Discord for extra tips.
I found a bug. How do I report it?
Open an issue on our GitHub Issues page with as much detail as possible.
